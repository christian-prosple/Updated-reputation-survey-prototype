## 2. Add AI/custom question builder slot

I still want the flexibility of describing a custom question in Replit chat and having it become part of the survey editor.

Add a “Custom Question Builder” section inside the Survey Editor.

UX idea:
- Add button: “Create custom question”
- Opens a modal/drawer with:
  - Textarea: “Describe the question you want to create”
  - Dropdown: target page
  - Button: “Generate question JSON”
  - Button: “Insert into survey”
  - Advanced JSON editor

Important: this does not need to call an AI API. The main goal is to create a structured place where Replit Agent/chat can insert generated question JSON.

Implement it as a human/agent-friendly workflow:
- The textarea is where I can write something like:
  “Create a pairwise employer comparison question using recognized employers, 10 comparisons, allow skip, store exposure data.”
- The app should either:
  1. Generate a basic template from keywords, or
  2. Let Replit Agent populate the JSON manually into the advanced editor.

The output should be a valid question object matching the survey config schema.
